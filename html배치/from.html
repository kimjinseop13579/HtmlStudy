type = "text" 는 문자 숫자 또는 특수문자 입력가능


type = "password" 는 비밀번호 입력시 사용 모든텍스트가 점으로 표현


type="number" 는 숫자만 입력가능 min max step 설정가능


type="email" 는 이메일형식을 감지 할수있는 타입

type="color"는 색상을 선택할수있는타입


type="range"는 슬라이드 형태로 색, 음량, 밝기 등 조절


type="date"는 날짜 선택


type="time" 는 시간선택


type="month"는 연.월 선택


날자타입은 week 연.주차 , datetime-local 날자+시간

type="checkbox" 는 다중선택을 위한 타입

게임 영화 음악

type="radio"는 여러선택지 중에 하나만 선택

사과 오렌지

type="file" 는 파일 업로드를 위한 타입


type="hidden" 는 사용자에게 제공하는 목적이 아닌 페이지 유지, 관리, 데이터 활용 등 페이지처리를 위해 사용된다.
