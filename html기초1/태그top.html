많이 사용되는 태그

  • 목록을 표현하는태그
  • 순서가 표시되지않는 태그
header - 페이지 상단의 영역표현
footer - 페이지 하단의 영역표현
main - 페이지의 본문, 주요 컨텐츠
nav - 메뉴
strong - 텍스트강조(굵게)

입력테그 - 진짜 사용자들이 이용하는 사이트에 필수 태그

테이블 (표) 태그

table - 표 만들기 태그 tr - 테이블에서 행을 표현혀는 태그 td - 테이블에서 텍스트를 작성할수 있는 태그
th - 테이블의 필드명 넣을수 있는 태그 (번호, 제목, 작성일 등등) thead - td가 있는 영역 tbody - td가 있는 영역
